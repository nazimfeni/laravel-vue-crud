Laravel vue crud. Here I use laravel11, vue3, Inertia tailwindcss


#setup
1. git clone https://github.com/nazimfeni/laravel-vue-crud.git
2. create and update.env file
3. composer install
4. npm install
5. npm update
6. php artisan migrate
7. php artisan key:generate
8. php artisan serve
9. npm run dev
